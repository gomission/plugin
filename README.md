# gomission — official Claude Code plugins

Marketplace for the Mission product family.

## Add the marketplace

```
/plugin marketplace add gomission/plugin
```

## Plugins

- **[mission](./plugins/mission)** — the Trust Graduation gate for agentic work. Slash commands for today/week/month prompts, the executor agent, and the Mission MCP server.

More plugins from the Mission product family (Content Improver, Social Growth, Newsletter, Living Presskit) will be added here.
